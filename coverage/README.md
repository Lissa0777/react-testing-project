# 🎬 Movies & Comics - Carrito de Compras

Este proyecto es una simulación de una tienda de **películas y comics** con un sistema de **carrito de compras** desarrollado en **JavaScript**.

El objetivo del proyecto es implementar funcionalidades básicas de un carrito y aplicar **pruebas unitarias con Jest** para validar su correcto funcionamiento.

---

##  Funcionalidades

* Mostrar productos disponibles
* Agregar productos al carrito
* Eliminar productos del carrito
* Calcular el total de la compra

---

## 🛠 Tecnologías utilizadas

* HTML
* CSS
* JavaScript
* Node.js
* Jest (Testing)

---

## Pruebas unitarias
Este proyecto utiliza **Jest** para realizar pruebas unitarias.

### Ejecutar los tests
npm run test

### Generar reporte de cobertura
npm run coverage

Esto generará la carpeta:
coverage/

donde se puede visualizar el porcentaje de cobertura del código.
